Permutation in String

Given two strings s1 and s2, write a function to return true if s2 contains the permutation of s1. In other words, one of the first string's permutations is the substring of the second string.

Example 1:

Input: s1 = "ab" s2 = "eidbaooo"
Output: True
Explanation: s2 contains one permutation of s1 ("ba").

Example 2:

Input:s1= "ab" s2 = "eidboaoo"
Output: False

Note:

  1. The input strings only contain lower case letters.
  2. The length of both given strings is in range [1, 10,000].

class Solution(object):
    def checkInclusion(self, s1, s2):
        """

        :type s1: str

        :type s2: str

        :rtype: bool

        """
        k = len(s1)
        N = len(s2)
        if k > N: return False
       
        memo = dict()
        
        for c in s1:
            memo.setdefault(c, 0)
            memo[c] += 1
        
        chars = len(memo)
        
        # init the first window

        for i in xrange(k):
            c = s2[i]
            if c in memo:
                memo[c] -= 1
                if memo[c] == 0:
                    chars -= 1
                elif memo[c] == -1:
                    chars += 1
                if chars == 0: return True
     
        for i in xrange(1, N-k+1):
            out_c = s2[i-1]
            in_c = s2[i+k-1]
            
            if in_c == out_c: continue
            if out_c in memo:
                memo[out_c] += 1
                
                if memo[out_c] == 0:
                    chars -= 1
                elif memo[out_c] == 1:
                    chars += 1
            if in_c in memo:
                memo[in_c] -= 1
                if memo[in_c] == 0:
                    chars -= 1
                elif memo[in_c] == -1:
                    chars += 1
   
            if chars == 0: return True
        return False

Longest Substring Without Repeating Characters

Given a string, find the length of the longest substring without repeating characters.

Example 1:

Input: "abcabcbb"
Output: 3 
Explanation: The answer is "abc", with the length of 3.

Example 2:

Input: "bbbbb"
Output: 1
Explanation: The answer is "b", with the length of 1.

Example 3:

Input: "pwwkew"
Output: 3
Explanation: The answer is "wke", with the length of 3. 
             Note that the answer must be a substring, "pwke" is a subsequence and not a substring.

Solution 0

class Solution(object):
    def lengthOfLongestSubstring(self, s):
        """

        :type s: str

        :rtype: int

        """
        memo = set()
        N = len(s)
        maxlen = 0
        start = 0
        l = 0
        for i in xrange(N):
            # ends on i

            c = s[i]
            l += 1
            if c not in memo:
                maxlen = max(maxlen, l)
                memo.add(c)
            else:
                while start < i:
                    c0 = s[start]
                    l -= 1
                    memo.remove(c0)
                    start += 1
                    if c == c0: break
                memo.add(c)
        return maxlen

Minimum Window Substring

Given a string S and a string T, find the minimum window in S which will contain all the characters in T in complexity O(n).

Example:

Input: S = "ADOBECODEBANC", T = "ABC"
Output: "BANC"

Note:

  • If there is no such window in S that covers all characters in T, return the empty string "".
  • If there is such window, you are guaranteed that there will always be only one unique minimum window in S.

Solution 0

  • 双指针，start, end. end 每次往前走一步
  • 先维护一个合格的 window
  • i 维护 window 的结尾
  • 动态推进 start
class Solution(object):
    def minWindow(self, s, t):
        """

        :type s: str

        :type t: str

        :rtype: str

        """
        n = len(s)
        k = len(t)
        if k > n: return ""
        
        # init memo of frequency of chars in t

        memo = dict()
        for c in t:
            memo[c] = memo.get(c, 0) + 1
        
        res = ""
        
        start = 0
        # init the first window

        counts = len(memo)
        windowsize = 0
        for i in range(n):
            c = s[i]
            # eat current char

            if c in memo:
                memo[c] -= 1
                if memo[c] == 0:
                    counts -= 1
                # shrink the size

                if counts == 0:
                    while start < i:
                        cs = s[start]
                        if cs not in memo:
                            start += 1
                            continue
                        if memo[cs] < 0:
                            start += 1
                            memo[cs] += 1
                        else:
                            break
                    if (not res) or i - start + 1 < len(res):
                        res = s[start:i+1]
        return res

Sliding Window Maximum

Given an array nums, there is a sliding window of size k which is moving from the very left of the array to the very right. You can only see the k numbers in the window. Each time the sliding window moves right by one position. Return the max sliding window.

Example:

Input: nums = [1,3,-1,-3,5,3,6,7], and k = 3
Output: [3,3,5,5,6,7] 
Explanation: 

Window position                Max
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7

Note: You may assume k is always valid, 1 ≤ k ≤ input array's size for non-empty array.

Follow up: Could you solve it in linear time?

Solution 0

  • 用 priority_queue
class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        std::priority_queue<int, std::vector<int>> queue;
        std::vector<int> res;
        if (nums.empty()) return res;
        
        if (k > nums.size() || k == 0) return res;
        
        std::unordered_map<int,int> set;
        // init the first window
        for (int i = 0; i < k; i++) {
            set[nums[i]] ++;
            queue.push(nums[i]);
        }
        
        res.push_back(queue.top());
        for (int i = 1; i < nums.size()-k+1; i++) {
            int out = nums[i-1];
            int in = nums[i+k-1];
            set[in]++;
            set[out]--;
            queue.push(in);
            
            int top = queue.top();
            while(!set.count(top) || set[top] == 0) {
                queue.pop();
                top = queue.top();
            }
            res.push_back(top);
        }
        return res;
    }
};
